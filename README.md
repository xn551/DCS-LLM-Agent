# DCS-LLM-Agent
This project aims to merge the LLM Agent technology to into the DCS World Mission script building. Now, you just describe the mission's key point, then the LLM would create the mission file automatically for you. 

## Progress
- the dcs_in_air.py is our start point, with pydcs we could build a mission file where the F15_c is started in the air and preparing for landing the airport.This file has been tested and worked in the newest DCS editon 2.9.25.2113.
